## Commands

- `npm run dev` — dev server
- `npm run build` — production build
- `npm test` — test
- `npm run lint` — linting

## Architecture

- React + TypeScript + Tailwind CSS + shadCn
- Strict TypeScript, no `any`

## Target

- App meteo
- dati fetchati da API
- Geolocalizzazione (su permesso) con dati presi dal dispositivo
- possibilità di aggiungere luoghi preferiti

## Conventions

- Functional components + hooks
- Tailwind per lo styling, no CSS inline
- Commit in inglese, imperative mood
